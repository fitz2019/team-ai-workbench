# Continuous Learning v2

Use when the team explicitly wants project-scoped learning from repeated AI sessions.

## What It Is

An instinct-based approach that:

- observes repeated session patterns
- stores them at project or global scope
- promotes reusable instincts over time

## Why It Is Advanced

It changes the operating model:

- hooks observe activity
- memory needs active management
- confidence and promotion rules must be curated

Do not enable this before the team has a stable manual knowledge-capture habit.
