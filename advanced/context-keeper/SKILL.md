# Context Keeper

Use when the team wants structured project-level save and resume workflows across interrupted sessions.

## Best Fit

- long-running efforts with many stops and resumes
- teams that want compact session snapshots
- projects where raw chat scrolling is not enough

## What It Provides

- project registration
- save summaries
- resume snapshots
- lightweight project memory management

## Why It Is Advanced

This introduces a separate memory-management workflow.

That is useful, but it should be an explicit choice rather than a default baseline.
