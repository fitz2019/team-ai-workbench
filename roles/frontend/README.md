# Frontend Role

Production-ready role pack for React and modern frontend projects.

Includes:

- frontend-specific `.agents` guidance
- frontend Codex role overlays
- ECC-derived frontend skills
- React-focused review and build-resolution roles

Use this role when the target repository contains:

- React or Next.js UI code
- component work
- client-side data fetching
- accessibility-sensitive interfaces
- frontend build and hydration risks
