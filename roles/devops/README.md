# DevOps Role

Production-ready role pack for deployment, CI/CD, operational safety, and infrastructure-facing troubleshooting.

Includes:

- DevOps-specific `.agents` guidance
- DevOps Codex role overlays
- ECC-derived deployment and ops skills

Use this role when the target work is about:

- CI/CD
- deployment and rollback
- infra review
- GitHub operational workflows
- network or environment troubleshooting
