# QA Role

Production-ready role pack for test and quality-assurance workflows.

Includes:

- QA-specific `.agents` guidance
- QA Codex role overlays
- regression, browser, and E2E skills
- ECC-derived `e2e-runner` role

Use this role when the target repository needs:

- regression protection after AI-assisted changes
- browser-based smoke and interaction checks
- Playwright-style E2E structure
- release verification and ship/no-ship decisions
